# Política de Seguridad - Atlas-Logistics

Este repositorio maneja lógica crítica de gestión de carga y orquestación de procesos.

## Reporte de Vulnerabilidades

Por favor, utiliza la funcionalidad **Private Vulnerability Reporting** de GitHub en la pestaña Security.

## Prácticas de Seguridad

- **Secretos:** Nunca subas archivos `.env` o credenciales de Zeebe/Postgres.
- **Infraestructura:** Usa el principio de mínimos privilegios para los contenedores Docker.
- **Auditoría:** Auditorías semanales mediante Dependabot y scripts locales de mantenimiento.
